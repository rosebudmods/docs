skin overrides supports many versions of minecraft, going back to 1.14.4. versions before 1.19.4 do not have the ui, and the mod can only be used by [modifying the files yourself](/skin-overrides/advanced/). from 1.19.4 onwards, the mod has both the [user interface](/skin-overrides/usage/) and [networking abilities](/skin-overrides/networking/).

here are the updates the mod fully supports:

| version | drop name          | since | estimated EOL |
| :-----: | :----------------- | :---: | :-----------: |
| 1.21.5  | Spring to Life     | 2.3.0 |    Q1 2028    |
| 1.21.4  | The Garden Awakens | 2.2.2 |    Q4 2027    |
| 1.21.3  | Bundles of Bravery | 2.2.0 |    Q3 2027    |
| 1.21.1  | Tricky Trials      | 1.0.0 |    Q2 2027    |
| 1.20.6  | Armored Paws       | 1.0.0 |    Q1 2027    |
| 1.20.4  | Bats and Pots      | 1.0.0 |    Q4 2026    |
| 1.20.2  | --                 | 1.0.0 |    Q3 2026    |
| 1.20.1  | --                 | 2.1.0 |    Q2 2026    |

the "estimated EOL" column shows approximately when that minecraft version will lose support (assuming 4 drops per year).
