skin overrides works with all versions from 1.18.1 onwards. for versions up until 1.19.3, there is no ui, and the mod can only be used by [modifying the files yourself](/skin-overrides/advanced/). here is a complete list of versions the mod supports:

every version of minecraft (except for versions which had hotfixes released) will have versions supporting it for at least 2 years after its release. the most used version of a particular update (the LTS version) will be supported for 5 years.
